# Hugo 博客客户端

目前做了mac版本。传送门https://github.com/jecshcier/blog_client_fe/releases

这里只是前端部分，使用webpack + react + codeMirror构建。

基于cherry框架，了解源码请移步https://github.com/jecshcier/cherry2



## 主要功能

- 文章管理（新建、删除、编辑文章）
- 分类标签管理（新建、删除分类、标签、关键字）
- 图片插入
- 代码片段插入
- 一键生成静态文件
